Iowa State University Medical Entomology Laboratory and Iowa Department of Public Health and the University Hygienic Lab (Iowa City, IA), monitors mosquito populations and mosquito-borne disease in the State of Iowa.

We will use shiny package to develop a web application to help entomologist to visualize, summarize and analyze the data produced by the surveillance program.

At this point we have written a shiny app based on part of the entire data set, we are using yearly mosquito counts for 8 sites and 20 years of data. However complete data has more sites and years on weakly basis. The shiny app have 4 tabset pannels each of them with a different plot, we describe each tabset with an example.

- 1) Tabset 1: Individual species description

The starting point is a figure for individual species. We can seelct only 1 species for this plot, and multiple sites. The resulting plot shows years in x axis and the mosquito species weight with facets on sites.

Choices on shiny: Species (only one) and Sites. 

d <- subset(msq.spyr, (specie == "Aedes.vexans") & (site == "Gvalley"))
ggplot(data = d, aes(x = year, y = prop.spst), color = site) + geom_point(size = 4) + 
    geom_line() + geom_line(aes(x = year, y = prop.spyr), color = I("red")) + 
    facet_wrap(facets = ~site, scales = "free")

As an example, we show here the proportion of Aedes.Vexan on Green Valley over time, the red line represent the average proportion of this speceis on each year. So on average across all years and sites, this speceis represent arround 75% of the mosquito counts. However on Green Valley the Aedes.Vexan proportion is always below the mean, What happen in Green Valley ? The Vexans don't like it anymore !

- 3) Third tabset: Comunity description

One important feature for the entomologist is to study the community characteristics, they are several indices reflecting different aspects of the mosquito commuinity composition. In this plot we can choose multiple indices and multiple sites to plot the yearly value of each index on each site. 

d <- subset(msq.ind.aux, (site %in% c("Gvalley", "Union")) & (Index %in% c("Simpson", 
    "SpeciesRichness")))
ggplot(data = d, aes(x = year, y = Index.val)) + geom_point(size = 4) + geom_line() + 
    geom_line(aes(x = year, y = Index.val), color = I("red")) + facet_grid(facets = Index ~ 
    site, scales = "free")

We have choose Green Valley and Union stations with SpeciesRichness and Simpson Index. Both measures are related to the diversity in the community.

- 4) Fourth tabset: Identify extreme comunities

Besides describing each community we may be interesting in comparing them, in particular we want to know if all communities are alike or there are some site-year observations that are different from the rest.

Working with species proportion within one site and year (instead of total counts) we compute an “average commuinity” wchich simply taking average of each species proportion. Then for each observations we compute the distance respec to that average community.

We plot a two pannel figure, on the right is the estimated density of the distances to the mid point community. The vertical red line represent the 90 quantile.

On the left panel there is a scatter plot of two comunity variables and we colored the points corresponding to communities further from the center community, this is with distance bigger than its 90% quantile. 

q <- quantile(msq$distout, probs = 0.9)
d <- data.frame(lax = msq[, "PrecipationMinus2"], lay = msq[, "DominanceBP"], 
    rare = as.factor(msq$distout > q))

p1 <- qplot(x = dat.den$x, y = dat.den$y, geom = "line", size = I(1.5)) + geom_vline(xintercept = q, 
    color = I("red")) + ylab("") + xlab("Distance to Average Community")

p2 <- qplot(data = d, lax, lay, color = rare) + scale_color_manual(values = c("black", 
    "red")) + xlab(as.character("PrecipationMinus2")) + ylab(as.character("DominanceBP")) + 
    theme(legend.position = "bottom")
grid.arrange(p1, p2, ncol = 2)

On the shiny it is posible to choose the quantile that determine wich communities are consider extreme and also the variables plot in the right pannel. In this example the selected quantile is 90% and the variables are Precipitation (horizontal) and Dominance index (vertical). Basically we can see all extreme communities shows big values in Dominance but are not extreme on precipitation.
